# GymHub

GymHub is a gym membership and fitness management system developed for the COM5009 Web Applications Development module.

## Features

### Member Features
- Register and login
- Purchase memberships
- Upgrade and downgrade memberships
- View class schedules
- Book classes
- Manage bookings
- Access fitness content

### Administrator Features
- Manage memberships
- Manage gym classes
- Manage bookings
- Manage fitness content
- View activity logs

## Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- Bootstrap
