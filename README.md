# GymHub

GymHub is a web-based gym membership and fitness management system developed for the COM5009 Web Applications Development module.

## Features

- User registration and login
- Membership management
- Membership upgrades and downgrades
- Class scheduling
- Class booking system
- Fitness content management
- Activity logging
- Role-based access control

## Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- Bootstrap
